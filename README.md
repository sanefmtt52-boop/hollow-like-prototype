# Hollow-like Prototype (Godot 4)

Прототип 2D-метроидвании в стиле Hollow Knight. Учебный проект «с нуля».

## Как открыть
1. Установить **Godot 4** (Standard, без .NET) — https://godotengine.org/download/windows/
2. Запустить Godot → в Project Manager нажать **Import**.
3. Указать файл `project.godot` из этой папки → **Import & Edit**.
4. Нажать **F5** (или ▶ вверху справа), чтобы запустить игру.

## Управление
A/D или стрелки — движение · Space — прыжок · J — атака · K — даш · L — лечение · F — действие.
(Даш и атака подключаются на следующих этапах.)

## Что уже сделано (M1)
- Движение и платформинг с «прощающими» таймингами (coyote time, jump buffer).
- Первый уровень с платформами-заглушками и камерой, следующей за игроком.
- Глобальное состояние `GameState` (здоровье, soul, способности) + HUD.

## Дальше (по плану)
- M2: бой гвоздём + Soul/лечение
- M3: враги, урон/смерть, скамейки-сохранения
- M4: способность-ключ (даш) + метроидвания-петля
- M5: сохранение на диск

Подробности — в [docs/game_design.md](docs/game_design.md).

## Структура
```
project.godot          настройки проекта (управление, autoload, главная сцена)
scenes/player/         игрок (сцена + скрипт)
scenes/levels/         уровни
scenes/ui/             интерфейс (HUD)
scripts/autoload/      глобальное состояние (GameState)
docs/                  дизайн-документ, кредиты ассетов
assets/                спрайты, звук, шрифты (пока заглушки в коде)
```
