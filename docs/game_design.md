# Дизайн-документ (черновик)

Прототип 2D-метроидвании в стиле Hollow Knight. Числа черновые — подбираем «на ощупь».

## Игрок
- Скорость бега: **300 px/s** (без инерции — мгновенный старт/стоп, стиль Mega Man X).
- Прыжок: `jump_velocity = -420`, гравитация `980`. Высота ~3 клетки.
- «Прощающие» тайминги: **coyote time 0.1с**, **jump buffer 0.1с**, переменная высота прыжка.
- Здоровье: **5 масок**. Урон врага: −1 маска + откидывание + 1с неуязвимости (мигание).
- **Soul**: 0–99. +11 за удар гвоздём. Лечение (Focus): удержать кнопку, стоит 33, лечит 1 маску.

## Гвоздь (атака) — милстоун M2
- Урон: 5. Короткий прямоугольный хитбокс перед игроком.
- Направления: вправо/влево (по взгляду), вверх, вниз.
- **Pogo**: удар вниз в воздухе по врагу/шипу отталкивает игрока вверх.
- Кулдаун удара: ~0.25с.

## Способность-ключ: даш — милстоун M4
- Рывок: `dash_speed 700`, длительность 0.18с, кулдаун 0.5с.
- Изначально **заблокирован** (`GameState.has_dash = false`), открывается пикапом.
- Открывает участок уровня, непроходимый без даша (метроидвания-петля).

## Враг «crawler» — милстоун M3
- Патрулирует платформу, разворот у края/стены. HP 10 (2 удара). Контактный урон.

## Скамейка — милстоун M3
- Взаимодействие (F): сохранить точку возрождения + полностью восстановить здоровье.

## Управление
| Действие | Клавиши |
|---|---|
| Влево / вправо | A / D или ← / → |
| Вверх / вниз (прицел удара) | W / S или ↑ / ↓ |
| Прыжок | Space |
| Атака | J |
| Даш | K |
| Лечение (Focus) | L |
| Действие (скамейка/пикап) | F |

## Не входит в первый прототип
Charms/амулеты, спеллы кроме лечения, карта мира, несколько уровней с переходами,
музыка/полировка, меню. Архитектура заложена так, чтобы добавлять это поверх без переделок.
