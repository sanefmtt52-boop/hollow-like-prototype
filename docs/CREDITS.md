# Кредиты и лицензии ассетов

Здесь записываем КАЖДЫЙ внешний ассет (спрайт, звук, шрифт) и его лицензию.
Это важно: даже бесплатные ассеты часто требуют указания автора, а для будущей
публикации нужно точно знать, что можно использовать.

Формат записи:

```
- Название: <что это>
  Автор: <кто>
  Источник: <ссылка>
  Лицензия: <CC0 / CC-BY 4.0 / ...>
```

## Рекомендуемые источники бесплатных ассетов
- **Kenney.nl** — паки 2D, лицензия CC0 (можно всё, атрибуция не обязательна, но приветствуется).
- **itch.io → Ansimuz** — красивый платформер-арт, часто бесплатно/CC0.
- **OpenGameArt.org** — фильтр по лицензии CC0/CC-BY.
- **freesound.org** — звуки (лицензию проверять у каждого файла).
